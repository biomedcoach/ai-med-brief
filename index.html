Apple's AI Image Codec PICO Achieves 1/3 File Size Reduction in Perceptual Compression Benchmark

Apple engineers developed PICO, a perceptual image compression codec designed to optimize file size while preserving human visual quality. Addressing challenges like slow entropy encoding, text artifact generation, and tile boundary inconsistencies, PICO incorporates innovations such as a one-shot context model, TextFidelityLoss, and TilingArtifactLoss. In a rigorous human subjective test involving 610 participants and 74,925 pairwise comparisons, PICO demonstrated a 30-50% reduction in file size compared to AV1, JPEG AI, and other codecs, while maintaining equivalent perceived quality. This advancement enables more efficient image sharing and storage, particularly on mobile devices, without compromising user experience, marking a significant step in AI-driven perceptual compression technology.

PICO achieves up to 50% smaller file sizes than AV1 and JPEG AI while maintaining human-perceived quality.
The codec was validated through a large-scale human subjective test involving 610 participants and 74,925 comparisons.
This breakthrough demonstrates AI's potential to optimize digital image handling for consumer devices, addressing the gap between mathematical compression metrics and human visual perception.
